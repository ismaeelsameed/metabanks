Core Banking Middleware

Products

MetaBanks Middleware is at the heart of all other MetaBanks solutions, it introduces an architectural layer on top of the Core Banking system in order to abstract the underlying complexity of the Core Banking business logic. The MetaBanks Middleware introduces a Service Oriented Architecture mechanism that exposes SOAP based web services that can be consumed over HTTPS. The MetaBanks Middleware ships with Software Development Kits (SDK’s) in multiple programming languages including VB.NET, C#, Java, PHP and Python that are ready to be used by IT teams to rapidly develop in house banking solutions without worrying about Core Banking implementation and integration details. A comprehensive list of features include:

{#
    #} {#
  • Architected and deployed as Stateless J2EE Enterprise JavaBeans.
    • #} {#
  • Scalable horizontally and vertically in order to process and handle millions of concurrent requests per second.
    • #} {#
  • Exposes a SOAP based interface with a well defined WSDL interface for enabling JAX-WS service consumption with WS-Security Profiles.
    • #} {#
  • Transport level encryption with a 2048 bit cipher to protect the communication channel between the Software Development Kit (SDK) and the MetaBanks Middleware.
    • #} {#
  • Message-level security to ensure confidentiality by digitally encrypting message parts; integrity using digital signatures; and authentication by requiring username, X.509, or SAML tokens.
    • #} {#
  • Provides SDK’s for various programming languages including Java, Python, PHP, VB.NET and C# to allow developers to easily retrieve data from the core banking system with minimal effort.
    • #} {#
  • Exposes numerous endpoints for retrieving data from the core banking system including but not limited to customer information, account information, account transactions, credit card information, loan information, Letter of Credit details, Letter of Guarantee details, Placements, Deposits, Foreign Exchange transactions, Interest Rate Swaps etc.
    • #} {#
  • Out of the box integration with the MetaBanks Web/Mobile Banking solution.
