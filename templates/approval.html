Approval Management System (AMS)

Products

MetaBanks Approval Management System (AMS) is a dynamic workflow engine designed to facilitate and automate the approvals process required for various banking activities. The system allows for various workflows to be setup and configured, the automated service extracts essential information needed for the approval process creating an efficient way to process transactions and reducing human error. The AMS enables approvers to take action at anytime directly from their email. It offers complete visibility into approval requests, approvers and comments. The AMS uses a business friendly user interface to manage workflows, user identities, access and service requests. In addition, the AMS exposes a web service that allows for other in-house banking solutions to integrate and request for approval processes to be initiated.

  • Setup and configure workflows based on approvals needed for each department activity.
  • Assign users to the workflow by querying Active Directory for the appropriate users.
  • Define each workflow process by participating approvers and level of authority needed.
  • Define input/information needed to complete each approval.
  • Audit trail allowing the audit department to overview and have a record of each approval.
  • In-house banking solutions can be easily integrated with AMS through a web service.
  • Automated email delivery to users defined in the workflow once the approval is initiated.
  • Initiates a callback to the application that originated the approval process with the approval results.
  • Rapidly generate Approval Transactions on demand for Filing reasons.
